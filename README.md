# File Organizer

A Python script that automatically sorts files in a folder into subfolders by type (images, documents, videos, audio, code, archives).

## Features
- Automatically detects file types by extension
- Creates category subfolders on the fly
- Handles duplicate filenames safely
- Prints a summary of what was moved

## How to run
```bash
python organizer.py
```
Enter the full path to any folder and it will be organised instantly.
